## Overview

The map recreates the real city. Key features of the settlement and its surroundings were taken into account during development. The city has been scaled down, but its proportions have been preserved.

The map is still under development. The city, nearby surroundings, and the ammunition depots in the south are complete. Future patches will continue to expand the map.

## Asset Use and Sources

We are ok with people using assets from this map. Some assets, including Sirte houses Blender models, are included in the sources.

If you want to extract textures used on Sirte, you can use [edds2png](https://github.com/wrdg/edds2png).
